## 2. Hero Talent Breakdown

### **Annihilator (Maintenance Focus)**
*Focuses on long-duration Void Metamorphosis windows.*
*   **Key Mechanic:** **Voidfall** stacks (build via Soul fragments, spent on *Reap*).
*   **Rotational Shift:** You must maintain **Void Ray** to pause the Fury drain. **Collapsing Star** should be used at maximum Soul Fragments (30) for peak efficiency.

### **Void-Scarred (Hybrid Focus)**
*Focuses on high-impact transformation windows and melee-weaving.*
*   **Key Mechanic:** **Voidsurge** procs.
*   **Rotational Shift:** Weave between melee builders and mid-range spenders. The goal is to trigger frequent, shorter Metamorphosis windows rather than one long one.

---

## 3. Full Rotational Priority

### **The "Void Maintenance" Loop (Meta Phase)**
1.  **Void Ray:** Use on cooldown (Priority #1) to pause Fury drain.
2.  **Collapsing Star:** Use at 30 Soul Fragments.
3.  **Reap / Cull:** Use at 3 stacks of **Voidfall**.
4.  **Devour:** Spend fragments to extend the window.

### **Building Phase (Non-Meta)**
1.  **Soul Immolation:** Maintain 100% uptime.
2.  **Fracture:** Generate Soul Fragments.
3.  **Reap:** Spend 3 stacks of **Voidfall**.
4.  **Void Metamorphosis:** Use at 100+ Fury.

## 4. Midnight 12.0.5 Changes (Audit Cleanup)
*   **NEW SPEC:** **Devourer** — This is a mid-range caster spec. Do not audit for Havoc/Vengeance melee priorities.
*   **CORE ABILITY:** **Void Ray** — This is a channeled ability. The bot should audit for **interruptions** or early cancellations during Meta.
*   **RESOURCE:** **Fury & Soul Fragments** — This spec uses both in a drain-management system similar to legacy Demonology Warlocks.
