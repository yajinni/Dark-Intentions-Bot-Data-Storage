## 2. Hero Talent Breakdown

### **Riders of the Apocalypse**
*Focuses on summoning the Four Horsemen and high mobility.*
*   **Key Mechanic:** **Rider's Champion** has a chance to summon a Horseman to assist you.
*   **Rotational Shift:** **Death Charge** replaces Death's Advance, allowing for 100% movement speed and mounted combat. Focus on extending Horsemen duration by spending Runic Power (**Fury of the Horsemen**).

### **San'layn**
*Focuses on Vampiric Strike procs and execute-style plague damage.*
*   **Key Mechanic:** **Vampiric Strike** replaces *Scourge Strike* procs, providing healing and a Haste buff (**Essence of the Blood Queen**).
*   **Rotational Shift:** During **Dark Transformation**, **Vampiric Strike** is guaranteed (**Gift of the San'layn**). Prioritize using these procs to erupt plagues via **Infliction of Sorrow**.

## 3. Full Rotational Priority

### **The Opener**
1.  **Outbreak:** Apply Virulent Plague.
2.  **Festering Strike (x2):** Generate initial **Lesser Ghoul** stacks.
3.  **Army of the Dead + Dark Transformation + Potion + Trinkets:** Activate major cooldowns.
4.  **Soul Reaper:** Apply early for execute/debuff overlap.
5.  **Putrefy:** Apply the new Shadow damage debuff.

### **Single-Target Priority**
1.  **Outbreak / Putrefy:** Maintain 100% Plague uptime.
2.  **Festering Scythe:** Use if the debuff has fallen off.
3.  **Soul Reaper:** Use if Health < 35% or if **Reaping** proc is active.
4.  **Death Coil:** Use if **Sudden Doom** proc active or Runic Power > 80.
5.  **Putrefy:** Use if Health > 35% and **Dark Transformation** is active.
6.  **Festering Strike:** Use if **Lesser Ghoul** stacks < 3.
7.  **Scourge Strike:** Use if **Lesser Ghoul** stacks ≥ 1 (Summons a Lesser Ghoul).

## 4. Midnight 12.0.5 Changes (Audit Cleanup)
*   **REMOVED: Festering Wounds** — The core mechanic of applying wounds to targets is gone.
*   **ADDED: Lesser Ghoul System** — You now build stacks on yourself (up to 8) with *Festering Strike* and spend them with *Scourge Strike* to summon temporary ghouls.
*   **ADDED: Putrefy** — A new baseline ability that deals Shadow damage and interacts with the execute phase.
*   **CHANGED: Soul Reaper** — Now only usable below 35% HP and consumes **Putrefy** charges for a massive burst.
*   **CHANGED: Forbidden Knowledge** — *Army of the Dead* now transforms *Death Coil* into **Necrotic Coil** and *Epidemic* into **Graveyard**.
*   **REMOVED: Abominable Limb** — No longer available for Unholy in the current Midnight build.
